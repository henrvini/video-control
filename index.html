INSTRUÇÕES DAS FUNÇÕES DOS BOTÕES

Mouse dentro do vídeo os controles aparecem

Mouse fora do vídeo os controles desaparecem

Botão 01 -> Back: retorna o vídeo

Botão 02 -> Forward: avança o vídeo

Botão 03 -> Back Time: diminui a velocidade de reprodução

Botão 04 -> Foward Time: aumenta a velocidade de reprodução

Botão 05 -> Start: inicia o vídeo

Botão 06 -> Stop: para a reprodução e retorna o vídeo para o início
